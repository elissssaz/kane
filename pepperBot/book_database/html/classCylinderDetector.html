Detailed Description

A ROS2 node that detects a cylindrical object based on laser scan data and publishes its coordinates.

Constructor & Destructor Documentation

◆ CylinderDetector()

CylinderDetector::CylinderDetector ( )

Constructor for the CylinderDetector class. It initializes parameters, subscribes to laser scan data, and sets up a TF2 transform listener.

< Diameter of the cylinder (default: 30 cm).

< Height of the laser (default: 0.75 meters).

< Minimum range of the laser (default: 0.5 meters).

< Maximum range of the laser (default: 5 meters).

< Allowed threshold for cylinder radius (default: 3 cm).

< Percentage of points that must match the expected cylinder pattern (default: 60%).

< Enables simulation time if running in a simulation environment.
